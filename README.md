# ai-window-remote-settings-telemetry-prompts

Expected directory structure

```
prompts/
  <prompt_name>/
    <major_version>/
      ├── config.json            # telemetry prompts metadata
      ├── outputSchema.json      # output schema definition
      ├── prompt.md              # Base prompt
triggers/
  <trigger_name>/
    <major_version>/
      ├── config.json 
```

For further details see [this document](https://docs.google.com/document/d/1BYzqoV3ArfP1XTjJwrEbKzXRDgRQMfcaG-RTHJwhx7s/edit?tab=t.etf3heqlbyzg)
